# optimization-methods
Laboratory works and coursework for the "Optimization Methods" course.

## Topics covered
- Gradient-based optimization methods
- Direct search methods (Hooke-Jeeves, Nelder-Mead)
- Evolutionary algorithms (genetic algorithms)
- Traveling Salesman Problem (heuristic and metaheuristic approaches)
- Performance comparison and analytical evaluation of algorithms

Each laboratory work includes:
- Problem formulation
- Algorithm implementation
- Experimental evaluation
- Analysis of results
